# Курсовая работа Smart Fridge

Готовый LaTeX-проект курсовой работы расположен в папке `XeLaTeX`.

Основной файл для компиляции:

```text
XeLaTeX/main.tex
```

Рекомендуемый компилятор Overleaf: `XeLaTeX`.

Стилевые файлы официального шаблона `timcw.sty` не изменялись. Файл `tim-logo.png` является технической заглушкой для титульного листа, так как команда шаблона `\cwtitle` ожидает наличие изображения с таким именем.
